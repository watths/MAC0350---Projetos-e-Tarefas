## Individual Project

This repository contains the individual project of the subject MAC0350 - Introdução ao Desenvolvimento de Sistemas de Software. The objective is to implement a website with the following requirements:

- Front-end (HTML, CSS and JS); <br>
- Back-end (FastAPI + HTMX); <br>
- Databases (SQL).

## E-mail

The code implements an email website, the user can:

- See received and sent e-mails;
- Write and send e-mails to other created users;
- Edit and delete sent e-mails;
- Reply e-mails;
- Search and navigate through e-mails.
